Branding

API Commons is all about being proud of our API design patterns, and when you publish your APIs into the commons, applying either CC BY-SA or CC0 licenses, we want to help the API community know. To help distinguish, openly licensed APIs, from everyone else, we want to introduce two separate icons, that can be used by an API provider who publishes their APIs to the commons:

API Commons will be working with search engines like APIs.io, and directories like Mashape and ProgrammableWeb to reference APIs that are published in the API Commons, with the appropriate icon as well.
