Add API

There are several ways you can submit your API definition or data model to the API commons. We want to make adding your API as easy as possible, no matter where your API definition is located.

Adding your API starts with generating an API Commons manifest for your API definition(s):

If your manifest is on github, and contains the term api-commons-manifest, it will automatically be indexed and included in API commons daily.

You can also tweet the URL of your manifest with the hashtag #apicommonsmanifest or email to apicommons@gmail.com and we will include in the commons.
